Tetravex

Android Puzzle Game

Tetravex is a single player 2D puzzle that was originally developed for windows systems to play in an internet browser The player is given a grid of tiles, typically 3x3. Each edge of the tiles has a different value or colour assigned to it. The aim of the game is to attempt to match each edge of the tiles to another edge that has the same value or number. Each puzzle is randomly generated, then the player must attempt to solve the puzzle in as little time possible.

Minimum O.S. - Android 5.0
