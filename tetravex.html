Tetravex

Android Puzzle Game

Tetravex is a single player 2D puzzle that was originally developed for windows systems to play in an internet browser. The player is given a grid of tiles, typically 3x3. Each edge of the tiles has a different value or colour assigned to it.

The aim of the game is to attempt to match each edge of the tiles to another edge that has the same value or number. Each puzzle is randomly generated, then the player must attempt to solve the puzzle in as little time possible.

Minimum O.S. - Android 5.0

Features

Game Modes

Classic - Complete the puzzle as quickly as possible.

Time Trial - Complete the puzzle before the timer runs out.

Arcade - Complete as many puzzles as possible in the given time-frame

Difficulty

The game implements algorithms that are designed to help adjust the difficulty of the game.

Unique Edges - Control the number of tiles with an edge that hasn't got a pair and must be placed along the perimeter.

Possible solutions - Control how many different solutions are possible.

Colour/Value Range - The range in different values is controlled by the difficulty settings

Options

Grid Size - Adjustable grid sizes, 2x2 to 5x5.

Colour Scheme - Option to disable or change the colour schemes
